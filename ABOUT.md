# About Clarity

## Project

**Clarity** is a Rust-native personal AI runtime that orchestrates LLMs, tools, and memory across every surface you use — terminal, desktop GUI, web IDE, headless scripts, and system tray. One agent core, multiple entry points.

Built for developers who want a single, local-first, fully auditable AI environment. No Node.js runtime. No Electron. No Python dependencies. No cloud lock-in.
