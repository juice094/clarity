## 📞 快速决策指南

**Q: 应该先实现哪个功能？**
> A: PersistentMemoryStore。当前 TUI 记忆不持久化是最大的可用性问题。

**Q: 后台任务为什么重要？**
> A: 它是子代理并行执行的基础设施，也是长时间任务的必需功能。

**Q: MCP 可以延后吗？**
> A: 可以。SubagentRunner 已提供工具扩展能力，8 个内置工具覆盖 80% 场景。

**Q: Web UI 什么时候做？**
> A: 建议暂缓。TUI 已足够验证核心功能，Web UI 投入大但边际收益递减。
